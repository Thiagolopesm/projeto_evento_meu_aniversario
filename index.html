Front-end

Front-end é a parte de um site ou aplicação que o usuário interage diretamente. É a camada visual e interativa, que inclui o layout, as cores, as fontes, os botões e tudo que pode ser visto na tela. O termo front-end também se refere à programação que torna isso possível, utilizando linguagens como HTML, CSS e JavaScript.

UI/UX

UI (User Interface) e UX (User Experience) são dois termos relacionados ao design de interfaces de usuário de sites, aplicativos e softwares em geral. Embora ambos estejam relacionados à experiência do usuário, eles se concentram em aspectos diferentes.

Data Science

Data Science é uma área interdisciplinar que combina estatística, programação, matemática e conhecimento de domínio para extrair insights de dados. É uma ciência que usa técnicas de análise de dados para extrair informações úteis de grandes volumes de dados, a fim de orientar a tomada de decisões.

Back-end

O back-end é a parte de um site, aplicativo ou software que fica "por trás das cortinas", ou seja, a parte que os usuários não veem diretamente. É responsável pelo processamento e armazenamento de dados, bem como pela comunicação com outros sistemas e serviços.
